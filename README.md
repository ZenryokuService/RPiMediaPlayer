# RPiMediaPlayer
ラズパイ上で動くJavaFX版のメディアプレーヤー

## YouTubeプレイヤー(Ver1.0)
### 処理フローV1
1. 起動
* プロパティファイル読み込み
* URLのリスト
* 再生時間(秒)
2. Java FX起動
* プロパティファイル読み込み
* フレーム作成
* 
3. URL読み込み
4. 次画面ロード
5. ロード無しで次画面表示
6. 終了
